## Description

Buildes Portal UI built by [NextJS](https://nextjs.org/) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ npm run dev

# build
$ npm run build

# start production mode
$ npm run start
```
